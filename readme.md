# Wordle CLI
## Features
- Custom solo game
- Daily game
- Online game
- Multi-language support
- Statistics

## Installation
### Client
#### Cloning the repository
- Click the Code button
- Download zip
- Unpack the file
- Go to client/
- `pip -m requirements.txt`
- `python3 main.py`
### Server
- Clone the repository
- Go to server/
- `pip -m requirements.txt`
- `flask run`
